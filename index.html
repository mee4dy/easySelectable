Options

Name Expected Default Description
item string li Selector of elements inside main block for selected
state boolean true Selectable on/off

Callbacks callback parameter el = current selected object;

Name parameter Description
onSelected el Executes after every selected set elements
onUnSelected el Executes after every unselected set elements
onSelecting el Executes after every change selected
